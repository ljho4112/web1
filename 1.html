HTML이란 무었인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.Web browsers receive HTML documents form a web server or form local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and orignally inclued cues for the appearance of the document.

HTML elements are the images and other objects, such as interactive forms, may be embedded into thr rendered page. It procides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
